# WEB103 Project 1 - *Smash Listicle*

Submitted by: **Tanvir Pulak**

About this web app: **This is a web app where users can view every single character from the Super Smash Bros. series, see what franchise they are from, learn which Smash game they debuted in, and read a short summary about each fighter.**

Time spent: **5** hours

## Required Features

The following **required** functionality is completed:

## Notes

One challenge I encountered was organizing all of the fighter data so that every character had the same shared attributes. I also had to make sure each character could link to its own detail page using a unique route.
